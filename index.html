    Q&A

    1. What is the three main diffrences between web1,web2 and web3 ?

      1. Unlike web1, in web2 and web 3 you can use crawlers to go over the web.
      2. While searching, in web1 you can only search for a certain url. in web2, you can use search engines to search the web with a specific query. In web3, the search is still happening with the help of the search engines, but, the search itself is much more progressive and using semantic search on your query.
      3. In web1, the pages are static, not as in web2 and web3, where the pages are dynamic, and can change often

    2. How is the search of the query "Medicine Side Effects" looks in web1, web2 and web 3?

      • Web1: No crawlers in web1, so you had to put as an input a url of a website to search in
      • Web2: The user will use search engines to search for the query he want to search, than he will get a list of links to websites containing words from his query
      • Web3: Same as web2, but the search can get as results link to a webpage (or specific part of it) that has a good realtion to the query asked
